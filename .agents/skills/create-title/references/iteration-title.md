# Title Iteration Mappings

Title-specific refinement mappings. Maps common user feedback to specific title modifications. Use alongside [iteration-core-copy.md](../../shared/references/iteration-core-copy.md) for the general iteration workflow.

## Text & Hook Changes

| User Says | Action |
|-----------|--------|
| "make it shorter" | Compress to under 50 characters. Cut secondary clauses, keep core hook. |
| "make it longer" | Add context, bracket modifier, or secondary clause. Stay under 100 chars (YouTube). |
| "more clickbait" | Add curiosity gap ("You Won't Believe..."), power words, brackets. Open a loop. |
| "less clickbait" / "more authentic" | Remove hype words. Use straightforward, honest framing. Lead with value. |
| "different hook" | Keep the same formula but change the opening element (different power word, different angle). |
| "more specific" | Add numbers, dollar amounts, timeframes, or named examples. |
| "more general" / "broader appeal" | Remove niche jargon, widen the target audience with universal language. |
| "change the text" / "different words" | Rewrite with synonyms and restructured phrasing while keeping the same formula. |

## Formula & Structure Changes

| User Says | Action |
|-----------|--------|
| "different formula" / "different approach" | Swap to a completely different template from [title-formulas.md](title-formulas.md). |
| "make it a question" | Convert to Question formula: "[Provocative Question]?" |
| "make it a list" / "use numbers" | Convert to Listicle formula: "[Odd Number] [Things]..." |
| "make it a how-to" | Convert to How-To formula: "How to [Result] in [Timeframe]" |
| "add brackets" | Append a bracket modifier: [2026], [FREE], [PROVEN], [STEP-BY-STEP] |
| "remove brackets" | Strip bracket modifiers while keeping the core title. |
| "make it contrarian" | Reframe as myth-busting or contrarian: "Why [Common Belief] Is Wrong" |

## Tone & Emotion Changes

| User Says | Action |
|-----------|--------|
| "more professional" | Remove urgency/hype. Use authority words (Proven, Expert, Data-Driven). Formal tone. |
| "more casual" / "more fun" | Add conversational language, contractions, slang appropriate to the niche. |
| "more urgent" | Add time pressure (Before It's Too Late, Last Chance, Now, Don't Miss). |
| "more exciting" | Add emotion words (Insane, Mind-Blowing, Epic, Game-Changing). |
| "calmer" / "toned down" | Remove superlatives, reduce emotional intensity, use understated language. |
| "more authoritative" | Frame as expert content: "The Complete Guide", "What Experts Know", "[Data-Backed]". |
| "funnier" | Add humor, wordplay, self-deprecation, or absurdity. |

## SEO & Keyword Changes

| User Says | Action |
|-----------|--------|
| "add keyword [X]" | Integrate the keyword naturally, preferably in the first 50 characters. |
| "front-load keyword" | Move the target keyword to the first 3-5 words of the title. |
| "more searchable" / "more SEO" | Add primary keyword early, use common search phrases, remove clever-but-unsearchable phrasing. |
| "remove keyword" | Strip the specified word while maintaining title coherence. |

## Platform Adaptation

| User Says | Action |
|-----------|--------|
| "make it work for Instagram" | Restructure as a caption first-line hook (under 125 chars, scroll-stopping). |
| "adapt for Twitter" | Compress to 70-100 chars, make it conversational, keep 1-2 hashtag budget. |
| "LinkedIn version" | Professional tone, thought-leadership framing, under 140 chars for above-fold. |
| "TikTok version" | Casual, direct, keyword-rich for TikTok search, front-load the hook. |
| "make it for all platforms" | Generate platform-adapted variants of the chosen title (V2-yt, V2-ig, V2-x, etc.). |

## Word Order Changes

| User Says | Action |
|-----------|--------|
| "put [X] first" | Move the specified element to the front of the title. |
| "swap the order" | Reverse or rearrange the main clauses. Word order dramatically affects CTR. |
| "start with the number" | Lead with the number if it's a listicle (e.g., "7 Ways..." instead of "The 7 Best Ways..."). |
| "start with the result" | Lead with the outcome/benefit instead of the method. |
