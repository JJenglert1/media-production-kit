# Description Anatomy by Platform

Platform-specific structures for descriptions and captions. Each platform has different conventions, character limits, and user expectations.

## YouTube Description Anatomy

The gold standard structure for YouTube video descriptions. Sections are ordered by priority — above-fold content is most critical.

```
[HOOK — 1-2 lines, above the fold]
Compelling value statement with primary keyword. This is what viewers see
before clicking "Show more."

[SUMMARY — 2-3 sentences]
Brief expansion on what the video covers. Secondary keywords here.
What will the viewer learn/gain/experience?

[TIMESTAMPS / CHAPTERS]
00:00 Introduction
02:15 First Major Topic
05:30 Second Major Topic
08:45 Key Takeaway or Demonstration
11:00 Summary & Next Steps

[RESOURCES & LINKS]
📌 Resources mentioned in this video:
- [Resource 1]: [YOUR LINK]
- [Resource 2]: [YOUR LINK]
- Free download/template: [YOUR LINK]

[CALL TO ACTION]
👉 Subscribe for weekly [topic] content: [CHANNEL LINK]
🔔 Turn on notifications so you never miss a video

[SOCIAL / CONNECT]
📱 Follow me on [Platform]: [YOUR LINK]
💬 Join the community: [YOUR LINK]

[ABOUT — boilerplate]
About [CHANNEL NAME]:
[1-2 sentence channel description for new viewers]

[HASHTAGS]
#Keyword1 #Keyword2 #Keyword3
```

### YouTube Rules
- Above the fold: First 150-200 characters. MUST contain hook + primary keyword.
- Timestamps: Start at `00:00`, minimum 3, ascending. Use keyword-rich titles.
- Hashtags: 3-5 at the end. First 3 appear above the video title.
- Links: Full URLs (YouTube doesn't support markdown links).
- Don't start with: "In this video..." — lead with value.
- SEO: Primary keyword in first sentence, 2-3 semantic variations throughout.

### YouTube Shorts Description
- Shorter format — 2-3 sentences max
- Hashtags in the title (not description)
- CTA: "Follow for more" or "Part 2?"
- No timestamps needed

## Instagram Caption Anatomy

```
[HOOK LINE — under 125 chars, above the fold]
The one line that stops the scroll. Must stand alone.

[BODY — value, story, or context]
2-5 short paragraphs with line breaks.
Each paragraph is 1-2 sentences max.

Use emojis as visual anchors ✨
Break up the wall of text.

[CTA — engagement prompt]
💬 What's your [related question]? Drop it below!
📌 Save this for later

[HASHTAGS — at end or first comment]
#Hashtag1 #Hashtag2 #Hashtag3 #Hashtag4 #Hashtag5
```

### Instagram Rules
- Above the fold: ~125 characters before "...more"
- Hashtags: 3-5 recommended (max 30, trending toward 5 cap)
- No clickable links — direct to "link in bio"
- Line breaks: Essential for readability. Use them liberally.
- Emojis: Acceptable and encouraged as visual punctuation
- Mix lengths: 60% short (<150 chars), 30% medium, 10% long

## TikTok Caption Anatomy

```
[HOOK PHRASE — front-loaded, keyword-rich]
Compelling first line that works with the video content.

[CONTEXT — brief]
1-2 sentences expanding on the hook.

[CTA]
Follow for Part 2 👆 | Save this 📌 | Comment your [X]

[HASHTAGS — inline or end]
#NicheTag #BroadTag #ContentSpecificTag
```

### TikTok Rules
- Search engine behavior: TikTok users search like Google. Keywords matter.
- Keyword alignment: Hashtags should match words spoken in audio AND caption text
- Above the fold: Must tap "see more" — front-load the compelling content
- Hashtags: 3-5 strategic (up to 9 with 3-3-3 strategy)
- Keep it concise — don't fill the 4,000-character limit with filler

## Twitter/X Post Anatomy

```
[HOOK + VALUE — complete message in 280 chars]
The entire post must stand alone. Lead with the insight or hook,
add value or context, end with 1-2 hashtags.

[OPTIONAL: Thread format for longer content]
1/ [Hook — strongest insight or claim]
2/ [Context or backstory]
3/ [Key point 1]
4/ [Key point 2]
5/ [Key point 3]
6/ [Summary + CTA]

🔗 [Link if applicable]
#Hashtag1 #Hashtag2
```

### Twitter/X Rules
- Full display: 280 characters visible (no truncation for standard tweets)
- Optimal length: 70-100 characters for highest engagement
- URLs: Always count as 23 characters regardless of length
- Hashtags: 1-2 maximum. Mid-tweet or end. NEVER start with a hashtag.
- Threads: 4-8 tweets, numbered (1/), hook in tweet 1, CTA in last tweet

## LinkedIn Post Anatomy

```
[HOOK — under 140 chars, above "See more"]
Bold opening that stops the professional scroll.
Contrarian, data-driven, or story-driven.

[LINE BREAK]

[BODY — thought leadership or story]
Short paragraphs.

One sentence per line works well on LinkedIn.

Use data and specifics:
→ "Revenue increased 340% in 6 months"
→ "We went from 3 to 47 enterprise clients"

Bullet points and arrows (→) for structure.

[TAKEAWAY]
The key insight or lesson.

[CTA]
♻️ Repost if this resonated
💬 What's your take? I'd love to hear in the comments.
👉 Follow [YOUR NAME] for more [topic] insights

#PascalCaseTag1 #PascalCaseTag2 #PascalCaseTag3
```

### LinkedIn Rules
- Above the fold: ~110 chars (mobile), ~140 chars (desktop) before "See more"
- Formatting: Single-sentence paragraphs, heavy line breaks, arrows (→)
- Tone: Professional but human. Thought-leadership preferred.
- Hashtags: 3-5 at bottom, PascalCase mandatory
- Links: Consider putting links in first comment (posts with links may get lower reach)
- Optimal length: 150-300 words

## Cross-Platform Adaptation Notes

When adapting a description from one platform to another:

| From → To | Key Changes |
|-----------|-------------|
| YouTube → Instagram | Drop timestamps, shorten dramatically, add hashtags at end, remove links |
| YouTube → Twitter/X | Compress to 1-2 sentences, keep core hook, max 2 hashtags |
| YouTube → LinkedIn | Professional tone, single-sentence paragraphs, PascalCase hashtags |
| YouTube → TikTok | Keep hook, shorten body, keyword-rich, 3-5 hashtags |
| Instagram → YouTube | Expand with timestamps, links, about section, SEO keywords |
| LinkedIn → Twitter/X | Compress to core insight, casual tone, shorter hashtags |
