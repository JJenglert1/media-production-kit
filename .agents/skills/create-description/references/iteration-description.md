# Description Iteration Mappings

Description-specific refinement mappings. Maps common user feedback to specific description modifications. Use alongside [iteration-core-copy.md](../../shared/references/iteration-core-copy.md) for the general iteration workflow.

## Hook & Above-Fold Changes

| User Says | Action |
|-----------|--------|
| "better hook" / "stronger opening" | Rewrite first 1-2 lines using a different hook technique (curiosity gap, bold claim, question, statistic). |
| "less boring opening" | Replace "In this video..." or generic openings with a pattern-interrupting first line. |
| "more intriguing" | Add curiosity gap or open loop to the above-fold text. |
| "more direct" | Lead with the core benefit/value immediately. Remove buildup. |
| "different angle" | Reframe the hook — same topic, different emotional entry point. |

## Structure & Length Changes

| User Says | Action |
|-----------|--------|
| "shorter" / "too long" | Cut non-essential sections. Prioritize: hook > CTA > body. Remove boilerplate. |
| "longer" / "more detail" | Expand body with additional context, benefits, or sections. Add resources section. |
| "add timestamps" | Insert chapter markers starting at `00:00`, min 3, ascending, keyword-rich titles. |
| "remove timestamps" | Strip timestamp section while keeping other content intact. |
| "fix timestamps" / "update timestamps" | Adjust chapter times/titles to match actual video content. |
| "better structure" | Reorganize with clearer section breaks, headers, and logical flow. |
| "add bullet points" | Convert paragraph content to scannable bullet list format. |
| "more sections" | Add missing sections (resources, about, social links, timestamps). |

## Tone & Style Changes

| User Says | Action |
|-----------|--------|
| "more professional" | Remove casual language, contractions, emojis. Use formal, authoritative tone. |
| "more casual" / "more personal" | Add conversational tone, contractions, personal anecdotes, emojis. |
| "more storytelling" | Restructure as narrative — add personal angle, before/after, journey. |
| "less salesy" | Soften CTAs, lead with value, remove promotional language. |
| "more promotional" | Strengthen CTAs, add urgency, highlight benefits more prominently. |
| "add emojis" | Insert emojis as visual anchors at section starts and key points. |
| "remove emojis" | Strip all emojis for a cleaner, more professional look. |

## SEO & Keyword Changes

| User Says | Action |
|-----------|--------|
| "more SEO" / "add keywords" | Integrate target keywords naturally in first sentence and throughout. Add semantic variations. |
| "less keyword-heavy" | Reduce keyword density, make language more natural and conversational. |
| "add keyword [X]" | Weave the specific keyword into the description naturally, preferably in the first 150 chars. |
| "optimize for search" | Front-load keywords, add question-based phrasing that matches search intent, use keyword-rich timestamps. |

## CTA Changes

| User Says | Action |
|-----------|--------|
| "different CTA" / "change the call to action" | Swap to a different CTA type (subscribe → comment → save → share). |
| "stronger CTA" | Make CTA more specific, add urgency or benefit ("Join 50K+ subscribers who..."). |
| "softer CTA" | Reduce pressure — use question-based or community-oriented CTAs. |
| "add subscribe CTA" | Insert a subscribe prompt with benefit framing. |
| "add link CTA" | Add a section directing to a specific resource with `[YOUR LINK]` placeholder. |
| "remove CTA" | Strip call-to-action elements (rare — usually better to soften than remove). |

## Platform Adaptation

| User Says | Action |
|-----------|--------|
| "make it for Instagram" | Restructure per Instagram anatomy: hook under 125 chars, short paragraphs, hashtags at end, no links. |
| "adapt for LinkedIn" | Professional tone, single-sentence paragraphs, PascalCase hashtags at bottom, thought-leadership angle. |
| "TikTok version" | Shorten to hook + 1-2 sentences + CTA + hashtags. Keyword-rich for TikTok search. |
| "Twitter version" | Compress to 280 chars max. Core hook + value + 1-2 hashtags. |
| "make it for all platforms" | Generate platform-adapted variants: VX-yt, VX-ig, VX-li, VX-tt, VX-x. |

## Content Changes

| User Says | Action |
|-----------|--------|
| "add [topic/section]" | Insert the requested content in the appropriate position within the description structure. |
| "remove [section]" | Delete the specified section while maintaining flow. |
| "mention [product/tool/resource]" | Add a reference with `[YOUR LINK]` placeholder in the resources section. |
| "add about section" | Append a channel/creator about section at the bottom. |
| "add social links" | Insert a social media links section with platform placeholders. |
