# Hashtag Iteration Mappings

Hashtag-specific refinement mappings. Maps common user feedback to specific hashtag set modifications. Use alongside [iteration-core-copy.md](../../shared/references/iteration-core-copy.md) for the general iteration workflow.

## Tag Swaps & Additions

| User Says | Action |
|-----------|--------|
| "add [specific hashtag]" | Add the requested tag to the set, removing the least relevant existing tag if at platform limit. |
| "remove [specific hashtag]" | Remove the tag and optionally replace with a similar alternative. |
| "swap [tag A] for [tag B]" | Direct replacement. Verify new tag is PascalCase and relevant. |
| "more niche tags" | Replace 1-2 broad/medium tags with smaller, more targeted niche tags. |
| "more broad tags" | Replace 1-2 niche tags with higher-volume broad tags for wider reach. |
| "add trending tags" | Research currently trending tags in the niche, swap in 1-2 replacing evergreen tags. |
| "remove trending tags" | Replace trending/seasonal tags with stable evergreen alternatives. |
| "add my brand hashtag" | Include the user's branded hashtag, typically replacing the least relevant niche tag. |

## Strategy Changes

| User Says | Action |
|-----------|--------|
| "more discovery" / "more reach" | Shift pyramid toward broad+medium tags. Add trending tags if available. |
| "more targeted" / "more engagement" | Shift pyramid toward niche+community tags. Prioritize <100K post-count tags. |
| "different strategy" | Switch variation type entirely (e.g., from Balanced to Trend-Heavy). |
| "mix trending and niche" | Create a 50/50 split between trending and niche tags. |
| "evergreen only" | Remove all trending/seasonal tags. Replace with stable, year-round relevant tags. |
| "fewer hashtags" | Reduce count to platform minimum while keeping the strongest performers. |
| "more hashtags" | Increase count toward platform maximum, maintaining pyramid balance. |

## Platform Adaptation

| User Says | Action |
|-----------|--------|
| "make it for [platform]" | Rebuild the set with platform-specific count, placement rules, and conventions. |
| "adapt for YouTube" | Reduce to 3-5 tags. Ensure the first 3 are the strongest (shown above title). |
| "adapt for Instagram" | 3-5 tags, end of caption placement, verify none are banned. |
| "adapt for TikTok" | Expand to 5-9 tags using 3-3-3 strategy, align with caption keywords. |
| "adapt for Twitter" | Reduce to 1-2 tags only. Choose the single most impactful + one niche tag. |
| "adapt for LinkedIn" | 3-5 tags, PascalCase, professional/industry terms, bottom of post. |
| "give me all platforms" | Generate a separate optimized set per platform from the chosen variation's keywords. |

## Content Alignment

| User Says | Action |
|-----------|--------|
| "match my title" | Extract core keywords from the user's title and ensure hashtags reinforce them. |
| "match my description" | Align hashtags with SEO keywords used in the description. |
| "different topic angle" | Shift hashtags to cover a different facet of the content topic. |
| "more specific to [subtopic]" | Narrow hashtags to focus on a specific aspect of the broader topic. |
| "broader topic" | Widen hashtags to cover the general category rather than specific subtopic. |

## Formatting & Presentation

| User Says | Action |
|-----------|--------|
| "put them in order" | Arrange by tier: broad → medium → niche → branded. |
| "alphabetical" | Sort alphabetically within tiers. |
| "just give me the copy-paste" | Present only the single copy-paste line, no breakdown. |
| "show me the breakdown" | Present with full tier labels and strategy explanation. |
| "combine V1 and V3" | Merge the strongest tags from both sets, respecting platform count limits. |
